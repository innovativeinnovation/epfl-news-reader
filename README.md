Overview
--------

Very basic EPFL News Progressive Web App client.

[Try the App](https://epfl-devrun.github.io/epfl-news-reader/)

Features
--------

  * Progressive Web App
    * Uses Service Worker to cache data and work offline
    * App Manifest for installing to homescreen
